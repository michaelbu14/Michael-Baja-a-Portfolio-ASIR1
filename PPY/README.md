# 👨‍💻 Módulo: Programación en Python (PPY)

## 🧠 Resumen del Módulo
Este módulo se ha centrado en el desarrollo de la lógica computacional y su posterior implementación en lenguaje Python. El aprendizaje se ha estructurado en dos fases críticas:
1. **Modelado y Algoritmia:** Representación visual mediante diagramas de flujo para planificar la lógica antes de su codificación.
2. **Implementación Técnica:** Traducción de algoritmos a código Python, dominando desde la sintaxis básica hasta la gestión de archivos y estructuras de control avanzadas.

---

## 🛠️ Competencias Adquiridas

### 1. Lógica y Diseño de Algoritmos (Diagramas de Flujo)
* **Qué se hizo:** Modelado de procesos lógicos utilizando diagramas de flujo (.vsdx).
* **Conceptos clave:** Representación visual de secuencias, estructuras condicionales y bucles. Esta etapa ha sido esencial para interiorizar la resolución de problemas antes de enfrentarme al entorno de desarrollo.

### 2. Fundamentos de Python y Tipado
* **Qué se hizo:** Desarrollo de scripts para comprender la sintaxis básica.
* **Competencias técnicas:** 
    * Convenciones de nombrado (`snake_case`, `PascalCase`, `camelCase`).
    * Tipos de datos: enteros, flotantes, booleanos y strings.
    * Manipulación avanzada de cadenas: *slicing*, métodos (`upper`, `strip`, `find`, `replace`) y formateo con f-strings.
    * Operaciones aritméticas y uso del módulo `math` (`ceil`, `floor`, `sqrt`, `round`).

### 3. Flujo de Control y Estructuras
* **Qué se hizo:** Implementación de lógica condicional y bucles para automatizar tareas.
* **Competencias técnicas:**
    * Estructuras condicionales: `if`, `elif`, `else` y `match-case`.
    * Operadores ternarios para una mayor eficiencia en el código.
    * Iteradores: Bucles `for` y `while` (incluyendo control con `break` y `else` en bucles).
    * Gestión de bucles infinitos y bucles anidados para procesamiento de datos.

### 4. Gestión de Archivos y E/S
* **Qué se hizo:** Interacción con el usuario y persistencia de datos.
* **Competencias técnicas:**
    * Lectura de datos por teclado (`input`) y conversión de tipos (`int`, `float`).
    * Escritura en archivos: Modos de apertura `'a'` (append) y `'w'` (write) mediante `with open()`.
